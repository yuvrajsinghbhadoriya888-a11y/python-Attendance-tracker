# 📋 Attendance tracker (Python)

## 📌 About the Project

This is a beginner-friendly Python project that marks attendance using roll numbers and stores the records in an Excel file. The program automatically creates a new column for the current date, prevents duplicate attendance entries for the same day, and validates user input to reduce errors.

I built this project to strengthen my understanding of Python fundamentals and gain hands-on experience working with Excel files using the `openpyxl` library.

---

## ✨ Features

* ✅ Marks attendance using roll numbers
* ✅ Stores attendance in an Excel sheet
* ✅ Automatically creates today's date column
* ✅ Prevents duplicate attendance for the same day
* ✅ Validates user input

   ```

   ```
   python main.py
